# H5特效祝福页面项目文档

## 🎯 项目概述
这是一个基于Lottie动画的H5特效页面，用户输入关键词触发全屏动画，支持商业化变现。

## 📁 项目结构
```
h5-lottie-effect/
├── index.html              # 主页面
├── animations/             # 动画文件目录
│   ├── cake.json          # 生日蛋糕动画
│   ├── money.json         # 金钱雨动画
│   ├── fireworks.json     # 烟花动画
│   └── rose.json          # 玫瑰花雨动画（高级特效）
└── README.md              # 项目说明
```

## 🚀 快速开始

### 1. 本地运行
```bash
# 进入项目目录
cd h5-lottie-effect

# 启动本地服务器（Python）
python -m http.server 8084

# 或者使用Node.js
npx serve .
```

### 2. 访问页面
打开浏览器访问：http://localhost:8084

### 3. 测试功能
- 输入"生日快乐" → 触发蛋糕动画
- 输入"恭喜发财" → 触发金钱雨动画
- 输入"烟花" → 触发烟花动画
- 输入"玫瑰花雨" → 需要解锁（模拟支付）

### 4. 测试解锁功能
按 `Ctrl+U` 可以模拟解锁全部特效（仅测试用）

## 🔧 功能说明

### 核心功能
1. **关键词触发**：输入特定关键词触发对应动画
2. **快捷按钮**：点击关键词按钮快速输入
3. **全屏动画**：Lottie动画全屏播放
4. **响应式设计**：适配手机和PC

### 商业化功能
1. **付费解锁**：点击"解锁全部特效"显示支付二维码
2. **本地存储**：解锁状态保存在localStorage
3. **客服联系**：支付后联系客服获取解锁

## 🎨 动画文件说明

### 当前使用的动画（示例）
1. **cake.json** - 生日蛋糕动画（简单矩形模拟）
2. **money.json** - 金钱雨动画（下落动画）
3. **fireworks.json** - 烟花动画（缩放动画）
4. **rose.json** - 玫瑰花雨动画（旋转下落）

### 替换为真实动画
**重要**：需要从LottieFiles下载真实动画文件替换：

1. 访问 https://lottiefiles.com
2. 搜索以下关键词：
   - "birthday cake" → 下载为 cake.json
   - "money rain" → 下载为 money.json
   - "fireworks" → 下载为 fireworks.json
   - "rose petals" → 下载为 rose.json
3. 替换 animations/ 目录下的文件

## 🌐 部署到GitHub Pages

1. 访问 https://github.com

### 步骤2：上传文件
1. 在仓库页面点击"Add file" → "Upload files"
2. 拖拽所有项目文件上传
3. 点击"Commit changes"

### 步骤3：开启GitHub Pages
1. 进入仓库 Settings
2. 左侧点击"Pages"
3. Branch选择"main"，文件夹选择"/ (root)"
4. 点击"Save"
