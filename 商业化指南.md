### 三、C端用户付费解锁方案

#### 方案A：简单收款码模式
- **定价**：9.9元/永久解锁
- **流程**：
  1. 用户点击"解锁全部特效"
  2. 弹出微信收款码
  3. 用户支付并截图
  4. 用户添加客服微信发送截图
  5. 客服发送解锁码或解锁链接

#### 方案B：付费链接系统（进阶）
- **技术实现**：
  ```javascript
  // 生成唯一解锁链接
  function generateUnlockLink(userId) {
    const token = md5(userId + 'secret_key');
    return `https://yourdomain.com/unlock.html?token=${token}`;
  }
  ```
- **流程**：
  1. 用户支付
  2. 系统自动发送解锁链接
  3. 用户点击链接自动解锁
